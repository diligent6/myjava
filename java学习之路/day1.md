# HelloWorld

1. 创建一个文件，存放代码

2. 新建一个java文件

   * 文件后缀名为.java
   * Hello.java(文件名)
   * **[注意点：]** 系统可能没有显示文件后缀名，需要手动打开显示文件后缀名

3. 编写代码

   ```java
   public class Hello{	
   	public static void main (String[]args){	
   			System.out.print("Hello,world!")		
   	}
   }
   ```

   4. 在控制台中 编译javac java文件名，会生成一个class文件
   5. 运行class文件，java class文件
